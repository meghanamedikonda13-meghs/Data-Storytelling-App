#  IPL Data Storytelling App

## Overview

The IPL Data Storytelling App transforms raw IPL data into meaningful stories and insights. It combines exploratory data analysis with visual storytelling to explain trends and patterns in IPL matches from 2008–2022.

## Features

* Dataset Introduction
* Exploratory Data Analysis (EDA)
* Interactive Visualizations
* Insights & Findings
* Recommendations
* Final Conclusion

## Dataset

IPL Matches Dataset (2008–2022)

File Used:
IPL_Matches_2008_2022.csv

## Technologies Used

* Python
* Pandas
* Plotly
* Streamlit

## Story Sections

1. IPL Growth Over Time
2. Most Successful Teams
3. Toss Strategy Analysis
4. Star Performers
5. Host Cities Analysis

## How to Run

Install dependencies:

python -m pip install -r requirements.txt

Run the application:

python -m streamlit run app.py
